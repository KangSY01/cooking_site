# 🍳 Cooking Site — 자취생을 위한 창의 요리 공유 플랫폼

Cooking Site는 **자취생·초보 요리 사용자**를 위한 레시피 공유 서비스입니다.  
사용자는 자신만의 창의적인 레시피를 올리고, 다른 사람의 레시피를 검색·평가하며 소통할 수 있습니다.  
본 프로젝트는 부산대학교 데이터베이스 수업의 팀/개인 프로젝트로 개발되었습니다.

---

## 🎯 프로젝트 목표

- 사용자가 레시피를 등록하고 공유할 수 있는 서비스 구축
- 태그, 재료, 난이도 등을 기반으로 레시피를 검색 및 필터링
- 댓글, 좋아요, 평점 등 사용자 참여 기능 제공
- PostgreSQL 기반 실제 데이터베이스 설계 및 API 서버 구현
- Django + DRF를 활용한 REST API 구축 경험

---

## 🧩 주요 기능

### ✔️ 레시피 기능
- 레시피 목록 조회
- 레시피 상세 페이지
- 재료, 조리 단계, 태그 포함
- 레시피 등록 / 수정 / 삭제 (추가 예정)

### ✔️ 사용자 기능
- 회원가입 및 로그인 (JWT 추가 예정)
- 나의 레시피 관리
- 댓글 작성 및 삭제
- 좋아요, 평점, 북마크 기능 (추가 예정)

### ✔️ 검색 및 추천
- 태그 기반 검색
- 재료 기반 검색
- 사용자의 선호도 기반 추천 기능 (확장 가능)
